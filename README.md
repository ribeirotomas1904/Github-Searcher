## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

- Node.js
- Yarn

### Installation

1. Clone the repo
```sh
git clone https://github.com/ribeirotomas1904/github-searcher.git
```

2. Install NPM packages
```sh
yarn install
```

3. Run the app in localhost
```sh
yarn start
```
